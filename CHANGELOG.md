## [0.1.0] - YYYY-MM-DD

### Added
- PyQt6 chat interface with system tray integration
- Ollama integration with automatic model selection
- Agent framework with tool calling capabilities
- File operations, process management, web browsing tools
- Screen capture and keyboard/mouse automation
- Vision model integration for screenshot analysis
- Platform-specific system control (Windows/Linux/macOS)
- Security layer with permissions and audit logging
- LangChain bridge for external tool integration
- Cross-platform packaging with PyInstaller
